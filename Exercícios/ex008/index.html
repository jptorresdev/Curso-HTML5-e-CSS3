Nessa frase, temos um termo em itálico usando a tag <I> (não semântica)

Texto marcado

Podemos criar também um texto marcado usando a tag <MARK> (semântica)

E no outro parágrafo, temos outo texto marcado no final

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo

Texto deletado

Podemos marcar um texto deletado para indicar que ele deve ser lido, mas não considerado com a tag <del> (semântica)

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois com a tag <ins> (semântica)

Existe também o sublinhado com a tag <U> (não semântica)
